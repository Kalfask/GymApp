A gym management app for coaches and athletes.

## Features
- Coach: Add members, create workout programs, generate PDFs
- Athlete: Login, request programs, view workouts

## How to Run

1. Install dependencies:
```bash
   cd server
   npm install
```

2. Start server:
```bash
   node index.js
```

3. Open in browser:
   - Coach: `frontend/coach.html`
   - Athlete: `frontend/athlete.html`
